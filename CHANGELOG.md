# Changelog

All notable changes to v-star are documented in this file.

## [0.2.0] - 2026-03-27

### Added
- Risk measures package: VaR, CTE (Expected Shortfall), full risk reports
- Examples directory with runnable demos (quickstart, Monte Carlo risk, CSV valuation)
- Python bridge with Jupyter notebook for visualization
- GitHub Actions CI workflow
- CONTRIBUTING.md

### Changed
- Updated README with CI badge, examples section, and risk measures documentation

## [0.1.0] - 2026-01-01

### Added
- Core actuarial engine: rates, annuities, reserves, mortality tables
- Parallel CSV streaming with CensusRecord parsing
- Monte Carlo interest rate simulation (GBM)
- Present value and v-star discount factor calculations
- Duration and convexity calculations
- CLI with read, montecarlo, and bench subcommands
- JSON output support
- Comprehensive test suite with benchmarks
